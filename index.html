Sincronização JSONBin.io

Gratuito, sem tela de autorização. Configure uma vez e você e a Ket acessam os mesmos dados automaticamente.

Painel JSONBin.io → API Keys → Secret Key

Como configurar

  1. Acesse jsonbin.io e crie uma conta gratuita
  2. Clique em + New Bin e cole: {"fixed":[],"install":[],"cash":[]} → Create Bin
  3. Copie o Bin ID exibido após criar
  4. No menu lateral → API Keys → copie a Secret Key
  5. Cole os campos acima, clique em Testar e depois Salvar
  6. Mande o Bin ID + Secret Key para a Ket pelo WhatsApp
